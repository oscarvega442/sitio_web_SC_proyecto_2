# Validador de Contraseñas

## Descripción
Micro‑módulo web en HTML y JavaScript para validar contraseñas según reglas de seguridad.

## Funciones públicas
- validar()
- validarPassword(password, confirmPassword)

## Reglas
- Mínimo 8 caracteres
- Al menos una mayúscula
- Al menos un número
- Al menos un carácter especial
- Sin espacios
- Coincidencia entre contraseña y confirmación

## Cómo ejecutar
Abrir `index.html` en el navegador.
